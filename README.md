# 🩺 Principal Component Analysis on Diabetes Dataset

> Reducing dimensions and uncovering meaningful patterns through PCA.

## 📌 Overview

This project demonstrates Principal Component Analysis (PCA) on the Diabetes Dataset using Python. The objective is to reduce dimensionality while preserving the most important information for analysis and visualization.

The notebook includes:

* Data preprocessing
* Feature scaling
* Covariance matrix computation
* Eigenvalue and eigenvector analysis
* Principal component extraction
* Variance visualization
